Twilio の利用

iutwilio

Twilio を利用しテスト結果を電話やSMSで通知できます。

使い方

tools/twilio/iutwilio.py を使用します。
iutwilio は python で書かれていますので、python が利用可能な環境で実行してください。
実行にはテスト結果の xml と、Twilio のアカウント情報、発信先番号を指定します。

例）
iutwilio.py result.xml --call +810123456789 --account_sid=AC00000000000000000000000000000000 --auth_token=XXX --number=+815012345678

コマンドラインオプション

--account_sid SID Twilio アカウントの SID を指定します。
--auth_token TOKEN Twilio REST API の auth トークンを指定します。
--number NUMBER 発信元番号を指定します。
--url URL 発信内容の TwiML もしくは アプリケーション URL を指定します。
--call NUMBER 電話を発信します。NUMBER に発信先電話番号を指定します。
--sms NUMBER SMS を送信します。NUMBER に送信先電話番号を指定します。
--ini 設定を INI ファイルから読み込みます。[Twilio] account_sid,auth_token,number,sms_number
--dryrun 発信を行わず実行します。
-v,--version バージョンを表示します。
-h,--help ヘルプを表示します。
